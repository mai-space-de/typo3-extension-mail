Introduction
============

**Mai Mail** is a TYPO3 v12 extension that provides a robust email queue
management system with a backend module for monitoring and controlling
outgoing mail.

Features
--------

* **Mail Queue** – Queue emails for reliable, asynchronous delivery.
* **Backend Module** – Visual queue management, statistics, and compose UI.
* **IMAP/POP3 Inbox Viewer** – Browse incoming messages from the backend.
* **PSR-14 Events** – Hook into queue and send lifecycle events.
* **Scheduler Integration** – Process the queue automatically with the TYPO3 scheduler.
* **Priority & Scheduling** – Set per-message priority and future delivery times.
